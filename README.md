# OURcohol
> 술자리를 더욱 즐겁게 하고, 그 술자리를 기억하기 위한 서비스

✋ 더 이상 술만 마시는 술자리는 그만

📆 친구들과의 술자리를 캘린더 형태로 추억하자 

💡여러 유용한 술자리 꿀팁까지



## 기술스택

### Front-end

- Flutter

### Back-end

- Django&DRF, MySQL, AWS Elastic Beanstalk


### **Design** by 허준우

- Figma
